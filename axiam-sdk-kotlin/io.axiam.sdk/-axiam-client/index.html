AxiamClient

class AxiamClient : AutoCloseable

The AXIAM Kotlin SDK's public REST entry point (CONTRACT.md §1–§7, §9).

All auth/authz operations are suspend functions (coroutines) — the canonical names ARE the suspend forms (§1); there are no *Async twins. A caller needing a blocking form wraps a call in runBlocking.

Construction is only via builder: tenantId (a tenant slug or UUID string) is required (§5) — there is no default tenant. Owns exactly one RefreshGuard, one SessionState, and one JwksVerifier.

Conforms to CONTRACT.md §1–§7, §9–§11 (including §6.1 mTLS).
